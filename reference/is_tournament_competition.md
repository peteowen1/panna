# Check if competition uses tournament (year-only) format

Check if competition uses tournament (year-only) format

## Usage

``` r
is_tournament_competition(code)
```

## Arguments

- code:

  Character, the competition code

## Value

Logical, TRUE if competition uses year-only seasons

## Examples

``` r
if (FALSE) { # \dontrun{
is_tournament_competition("ENG")   # FALSE
} # }
```
