# Get all seasons since a start year

Returns a vector of season strings from start_year to current season.

## Usage

``` r
get_seasons_since(start_year = 2017)
```

## Arguments

- start_year:

  First season start year (default 2017)

## Value

Character vector of season strings (e.g., "2017-2018", "2018-2019", ...)

## Examples

``` r
if (FALSE) { # \dontrun{
get_seasons_since()  # From 2017
} # }
```
