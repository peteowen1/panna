# Check if league is available on Understat

Check if league is available on Understat

## Usage

``` r
is_understat_league(code)
```

## Arguments

- code:

  Character, the league code

## Value

Logical, TRUE if league is available on Understat

## Examples

``` r
if (FALSE) { # \dontrun{
is_understat_league("POR")
} # }
```
