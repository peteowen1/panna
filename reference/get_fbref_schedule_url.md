# Get FBref schedule URL

Constructs the FBref fixtures/schedule URL for a competition and season.

## Usage

``` r
get_fbref_schedule_url(code, season)
```

## Arguments

- code:

  Character, the league code (e.g., "ENG", "WC")

- season:

  Character, the season (e.g., "2023-2024" or "2024")

## Value

Character, the full FBref schedule URL

## Examples

``` r
if (FALSE) { # \dontrun{
get_fbref_schedule_url("WC", "2022")
} # }
```
