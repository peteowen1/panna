# Get players on pitch for a splint

Returns the player IDs for home and away teams in a splint.

## Usage

``` r
get_splint_players(splint_data, splint_id)
```

## Arguments

- splint_data:

  Combined splint data from create_all_splints

- splint_id:

  Splint identifier

## Value

List with home_players and away_players character vectors
