# Check if Coordinates are in Penalty Area

Check if Coordinates are in Penalty Area

## Usage

``` r
is_in_penalty_area(x, y, attacking = TRUE)
```

## Arguments

- x:

  X coordinate (0-100)

- y:

  Y coordinate (0-100)

- attacking:

  If TRUE, checks attacking penalty area (x \> 83)

## Value

Logical indicating if in penalty area
