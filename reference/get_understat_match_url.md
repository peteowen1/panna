# Get Understat match URL

Constructs the Understat match page URL for a given match ID.

## Usage

``` r
get_understat_match_url(understat_id)
```

## Arguments

- understat_id:

  Numeric or character, the Understat match ID

## Value

Character, the full Understat match URL

## Examples

``` r
if (FALSE) { # \dontrun{
get_understat_match_url(28988)
} # }
```
