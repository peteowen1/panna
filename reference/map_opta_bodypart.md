# Map Opta Events to Body Part

Map Opta Events to Body Part

## Usage

``` r
map_opta_bodypart(type_id, qualifiers = NULL)
```

## Arguments

- type_id:

  Vector of Opta type_id values

- qualifiers:

  Parsed qualifier information (optional)

## Value

Character vector of body parts: "foot", "head", or "other"
