# Get FBref competition ID

Looks up the FBref competition ID for a given league code.

## Usage

``` r
get_fbref_comp_id(code)
```

## Arguments

- code:

  Character, the league code (e.g., "ENG", "WC")

## Value

Integer, the FBref competition ID

## Examples

``` r
if (FALSE) { # \dontrun{
get_fbref_comp_id("WC")   # 1
} # }
```
