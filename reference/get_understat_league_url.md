# Get Understat league URL

Constructs the Understat league page URL for a competition and season.

## Usage

``` r
get_understat_league_url(code, season)
```

## Arguments

- code:

  Character, the league code (e.g., "ENG", "RUS")

- season:

  Character or numeric, the season year (e.g., "2024" or 2024)

## Value

Character, the full Understat league URL

## Examples

``` r
if (FALSE) { # \dontrun{
get_understat_league_url("ESP", "2023")
} # }
```
