# Map Opta Type ID to SPADL Action Type

Map Opta Type ID to SPADL Action Type

## Usage

``` r
map_opta_action_type(type_id, qualifiers = NULL)
```

## Arguments

- type_id:

  Vector of Opta type_id values

- qualifiers:

  Parsed qualifier information (optional)

## Value

Character vector of SPADL action types
