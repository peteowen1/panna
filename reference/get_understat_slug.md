# Get Understat URL slug

Looks up the Understat URL slug for a given league code.

## Usage

``` r
get_understat_slug(code)
```

## Arguments

- code:

  Character, the league code (e.g., "ENG", "RUS")

## Value

Character, the Understat URL slug

## Examples

``` r
if (FALSE) { # \dontrun{
get_understat_slug("RUS")  # "RFPL"
} # }
```
