# Get available Understat seasons

Returns a vector of available seasons for Understat data. Understat has
data from 2014 onwards.

## Usage

``` r
get_understat_seasons(start_year = 2014)
```

## Arguments

- start_year:

  First season year (default 2014)

## Value

Character vector of season years

## Examples

``` r
if (FALSE) { # \dontrun{
get_understat_seasons(2020)
} # }
```
