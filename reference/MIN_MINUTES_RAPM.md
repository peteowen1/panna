# Default minimum minutes for RAPM matrix inclusion

Players must have at least this many total minutes to be included in the
RAPM design matrix as individual players. Players below this threshold
are grouped into a "replacement" pool.

## Usage

``` r
MIN_MINUTES_RAPM
```

## Format

Integer value: 90

## Examples

``` r
MIN_MINUTES_RAPM
#> [1] 90
```
