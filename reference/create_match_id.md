# Create unique match ID

Generates a unique identifier for a match based on season, date, and
teams.

## Usage

``` r
create_match_id(season, date, home_team, away_team)
```

## Arguments

- season:

  Season string (e.g., "2023-2024")

- date:

  Date of match

- home_team:

  Home team name

- away_team:

  Away team name

## Value

Character vector of match IDs
