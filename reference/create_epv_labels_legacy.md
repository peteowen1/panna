# Deprecated: Create EPV Labels (Legacy Format)

**\[deprecated\]**

[`create_next_goal_labels()`](https://peteowen1.github.io/panna/reference/create_next_goal_labels.md)
for the current approach.

``` r
create_epv_labels_legacy(spadl_actions)
```

## Arguments

- spadl_actions:

  SPADL actions

## Value

Data frame with scores_this_possession and concedes_next_possession
