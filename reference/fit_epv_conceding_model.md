# Deprecated: Fit EPV Conceding Model

**\[deprecated\]**

[`fit_epv_model()`](https://peteowen1.github.io/panna/reference/fit_epv_model.md)
instead. The unified EPV model now handles both scoring and conceding
predictions using a multinomial approach.

``` r

```

## Arguments

- ...:

  Arguments passed to
