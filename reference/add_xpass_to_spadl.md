# Add xPass to SPADL Actions

Adds pass completion probability to pass actions in SPADL data.

## Usage

``` r
add_xpass_to_spadl(spadl_actions, xpass_model)
```

## Arguments

- spadl_actions:

  SPADL actions data frame

- xpass_model:

  Fitted xPass model

## Value

SPADL actions with xpass column added for passes
