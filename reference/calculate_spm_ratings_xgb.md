# Calculate SPM ratings using XGBoost model

Calculate SPM ratings using XGBoost model

## Usage

``` r
calculate_spm_ratings_xgb(player_features, spm_xgb_model)
```

## Arguments

- player_features:

  Data frame of player features

- spm_xgb_model:

  Fitted XGBoost SPM model from fit_spm_xgb

## Value

Data frame with SPM ratings
