# Get tournament years with available data

Returns years when tournaments were held (with FBref data available).

## Usage

``` r
get_tournament_years(code)
```

## Arguments

- code:

  Tournament code (e.g., "WC", "EURO", "COPA_AMERICA")

## Value

Character vector of years

## Examples

``` r
if (FALSE) { # \dontrun{
get_tournament_years("EURO")
} # }
```
